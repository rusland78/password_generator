# Генератор паролей
✔️ ❌
## Описание проекта
Программа генерирует заданное количество паролей и включает в себя умную настройку на длину пароля, а также на то, какие символы требуется в него включить, а какие исключить.

## Что умеет программа:
- настройка количества символов и их типов в паролях;
- генерация заданного количества паролей;
- проверка введённых значений и выдача сообщений об ошибках («защита от дурака»);
- определение «безопасности пароля» и вывод информационного сообщения;
- повторная генерация паролей, если необходимо.
